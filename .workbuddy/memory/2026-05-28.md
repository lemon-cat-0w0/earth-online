## 2026-05-28

### 部署故障排查与修复

- **Vite 版本升级**：Cloudflare Pages 要求 Vite ≥ 6.0，从 5.4 升到 6.4，同步更新 vite-plugin-pwa 到 0.21
- **添加 .gitignore**：排除 node_modules/、dist/、.wrangler 等
- **SPA 路由修复尝试**：
  - 首次尝试 BrowserRouter + `_redirects` → Cloudflare 报死循环/状态码错误
  - 最终方案：HashRouter（URL 带 `#`），纯前端路由，零服务器配置
- **Service Worker 缓存污染修复**：
  - iPhone Safari 激进缓存导致旧损坏 SW 残留
  - 改为手动注册：页面打开先卸载所有旧 SW → 延迟 300ms → 注册新 SW
  - `injectRegister: null` 禁止 vite-plugin-pwa 自动注入，避免时序冲突
- **wrangler autoconfig**：运行 `npx wrangler setup` 生成 wrangler.jsonc
  - 含 `"not_found_handling": "single-page-application"` 原生 SPA 支持
  - 新增 `npm run deploy` / `npm run preview` 脚本
